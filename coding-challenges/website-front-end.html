Website Front End

  I have built a fully responsive and mobile-friendly portfolio website using HTML, SASS and TypeScript, and used GitHub pages for its deployment.

  Being accustomed to another language with static typing (C#), I got used to TypeScript extremely easily. It allowed me to write cleaner code and helped me understand the errors you might come accross by using vanilla JavaScript. SASS is also another great tool that allowed me to keep my files organized while writing code faster.

The CSS and JavaScript of the website:

  • add responsiveness between all devices
  • add transition effects (color change, scale, item shadows, etc.) when hovering some items
  • allow the user to maximize images on click while preventing downloading them
  • offer smooth transitions between pages
